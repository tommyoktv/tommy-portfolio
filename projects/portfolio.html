Deployment

This website is deployed using GitHub Pages. The deployment process includes:

  • Push code to GitHub repository
  • Enable GitHub Pages in repository settings
  • Select main/master branch as source
  • Access via tommyoktv.github.io/portfolio

Alternative Deployment Options

  • Netlify (www.netlify.com)
  • Vercel (vercel.com)
  • Firebase Hosting (firebase.google.com)
